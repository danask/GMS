What is GMS?

Gardening Management System (GMS) is a system to notify the current status as alerts and recommend the action through data analysis with source data, which is about a variety of environment factors such as weather, temperature, humidity, and so on.

For monitoring system, we basically need to build the monitoring equipment with the sensors in the network. This data is sent to a main server to display and guide properly it using machine learning to end user. Finally, the main server filters more useful information to recommend and if neccessary, the notification is automatically sent to the user.

Sensor Data Collection

Raspberry-Pi

Raspberry Pi server monitors the target and capture the source data and image via the sensors.

Sensors

DHT (Temperature, Humidity sensor), PIR (motion sensor), and camera are required. These are working on Raspberry Pi with Python.

Firebase database and storage

Raspberry Pi server uploads the files and updates the data to database. It is neccessary to use the interface between server and Firebase database and storage.
