Welcome to GMS

GMS stands for Gardening Management System. This is an integrated solution program to provide ease of plant management with the environmental parameters such as temperature, humidity, and daily weather information. User can check the status on their web browser out and the system recommends an appropriate action as the situation. Therefore, this application helps people who are interested in the advanced management of gardening to monitor their plant more efficiently and flexibly. Moreover, this will be suitable for implementing and extending various programming language and network knowledge which we have learnt.

What is GMS?

Gardening Management System (GMS) is a system to check the status as alerts and take action like watering through data analysis with the collected environmental factors such as weather, temperature, humidity, and so on.

For monitoring system, we basically need to build the monitoring equipment with the sensors in the network. This data is sent to a main server to display the guideline using machine learning. If neccessary, the notification enables the user to use the system efficiently. (ver. 1)

For controllable system, LCD, step motor, RFID, buzzer, and LED are used and interact with users. User can control water sprinkler and alram control more efficiently using remote panel and mobile controller. These are more realistic and active implementation for users after monitoring. (ver. 2)

Sensor Data Collection and Controllable Devices

Raspberry-Pi

Raspberry Pi server monitors the target and capture the source data and image via the sensors.

Sensors and Devices

DHT (Temperature, Humidity sensor), PIR (motion sensor), and camera are required. These are working on Raspberry Pi with Python. 2-line LCD, step motor with control panel, RFID and tokens, buzzer active speaker, and LED lamp.

Python, MongoDB, Firebase database and storage

Raspberry Pi server uploads the files and updates the data to database through python programs. It is neccessary to use as the interface between server and database and storage.

Recommendation
