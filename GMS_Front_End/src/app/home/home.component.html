About GMS

Gardening Management System (GMS) is a system to notify the current status as alerts and recommend the action through data analysis with source data, which is about a variety of environment factors such as weather, temperature, humidity, and so on.

For monitoring system, we basically need to build the monitoring equipment with the sensors in the network. This data is sent to a main server to display and guide properly it using machine learning to end user. Finally, the main server filters more useful information to recommend.
